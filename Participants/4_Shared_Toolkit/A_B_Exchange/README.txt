A_B_Exchange - official Workflow A <-> Workflow B communications
================================================================

Create this same folder in your TEAM Google Drive (or Microsoft 365).
Markers and Buddies assess the cross-workflow exchange from that Drive folder.

Required contents (PDF exports of the Word templates):
  - Analysis Requests filed by Workflow A  (R-01, R-02, R-03...)
  - Findings Memos returned by Workflow B  (FM-01, FM-02, FM-03...)
  - Optional curated charts attached to memos

Templates:
  ../Templates/Analysis_Request.docx
  ../Templates/Findings_Memo.docx

Naming: TeamShortName_WSA_R01_Request_v1.pdf
        TeamShortName_WSB_FM01_Findings_v1.pdf

Also update the Request Log tab in GBF_Consulting_Toolkit.xlsx.
Chat / Zalo / Messenger is fine for logistics only - not the assessment record.
Minimum three completed R/FM pairs before final strategic options.
